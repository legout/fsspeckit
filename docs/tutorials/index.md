# Tutorials

Start here if you are new to fsspeckit. Tutorials are learning-oriented walks
through a complete task from start to finish.

## Local Dataset Lifecycle

[Local Dataset Lifecycle](local-dataset-lifecycle.md) is the recommended first
path for anyone using fsspeckit as a library. It is a single, copyable, offline
script that configures a local filesystem, writes a Parquet dataset, reads it
back, verifies the result, and cleans up a named sandbox. No cloud credentials
required.

Once you have run it, move on to the [How-to Guides](../how-to/index.md) for
specific tasks or the [Reference](../reference/api-guide.md) for the full API.
